iКак устроена очередь

🎟 Номер-талон закреплён
У каждого пациента свой номер (напр. C-007). Он выдаётся при отметке прихода и не меняется — даже если пациента пропустили или подвинули в очереди.
⏱ Очередь — по «времени к приёму»
Живая запись: по моменту прихода.
Запись по времени: по max(время записи, момент прихода) — раньше своего слота пациент не встанет.
🏃 Опоздал — встал по приходу
Пришёл позже своего времени — система ставит его по фактическому приходу, а не по записи. Те, кто пришёл раньше, идут первыми.
Порядок сортировки: 1 · «Срочно»→ 2 · по времени к приёму→ 3 · по номеру талона

РРесепшн

✓
Пришёл В очереди
Отметить приход пациента. Он встаёт в живую очередь, печатается талон с номером. Для записи по времени талон выдаётся именно в этот момент.
▶
Вызвать к врачу Идёт приём
Пациент уходит к врачу. На ТВ в холле загорается «приглашается № …» с кабинетом.
⏭
Пропустить → Вернуть Пропущен
Не отозвался — отметьте «Пропустить». Подошёл — «Вернуть в очередь»: номер сохраняется, но пациент встаёт в конец (по новому времени прихода).
⚡
Срочно
Поднять пациента над всей очередью. Важно: запись, чьё время ещё не наступило, перетаскиванием выше слота не сдвинуть — для этого есть только «Срочно».
⇅
Перетащить (drag)
Поменять местами тех, кто уже пришёл. Если перетащили, а порядок не изменился — значит у записи ещё не наступило время; используйте «Срочно».
✕
Отмена Отменён
Пациент ушёл / не придёт. Запись отменяется, слот освобождается, она исчезает из очереди, киоска и с ТВ.

ВВрач

▶
Начать приём Идёт приём
Открыть приём вызванного пациента. Одновременно может идти только один приём — закройте текущий, прежде чем начать следующий.
■
Завершить Завершён
Приём окончен. Слот укорачивается до фактического времени — хвост освобождается для других пациентов.
↩
Откатить (ошиблись кнопкой)
Только на своей записи: «Завершён» → «Идёт приём», «Идёт приём» → «В очереди». Ресепшн откатить приём не может.
i
Что видит врач
Свою живую очередь на сегодня: кто вызван, кто ждёт, номера талонов и примерное ожидание. Порядок — общий для всех экранов.
Талон + QR
На талоне номер и QR. Пациент сканирует → /t/КОД → открывается его личная страница статуса. Обновляется вживую.
Страница статуса
Пациент видит свой номер, позицию в очереди и примерное время ожидания. Меняется само, перезагружать не нужно.
Киоск и ТВ
Киоск: пациент сам берёт талон или отмечается по телефону. ТВ в холле: «сейчас приглашается № …» и список ожидания.

?Частые ситуации

Пациент опоздал на свою запись.
Просто нажмите «Пришёл». Система сама поставит его по времени прихода — раньше тех, кто ждёт, он не вклинится.
Не отозвался на вызов.
«Пропустить». Когда подойдёт — «Вернуть в очередь»: номер тот же, место — в конце.
Перетащил запись вверх — порядок не изменился.
У неё ещё не наступило время приёма. Поднять можно только кнопкой «Срочно».
Нужно принять вне очереди (срочный случай).
Кнопка «Срочно» — пациент поднимается над всеми, номер талона сохраняется.
Врач случайно нажал «Завершить».
Врач сам нажимает «Откатить» на своей записи — вернётся «Идёт приём».
Пациент передумал / ушёл.
«Отмена» — запись уходит из очереди, слот освобождается для других.
